var的问题#

  1. var声明的变量,声明到全局 污染全局变量 (函数作用域 全局)
  2. 变量提升 可以在声明之前调用 function var import
  3. let const 可以 {}方式来连用 块作用域
  4. var 能重复声明, 在同一作用域下
var a = 1;
{
var a = 2
}
console.log(a)
let a = 1;
{
console.log(a) // 暂存死区(Cannot access 'a' before initialization)
let a = 2
}
console.log(a)
const a = {} // 深度改变，只要不改变空间即可
a.x = 100
console.log(a)
